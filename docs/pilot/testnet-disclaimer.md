# Testnet Disclaimer — NextBlock Pilot

Read this before participating in the NextBlock pilot.

## This is a testnet

- The pilot runs on **Base Sepolia**, a public Ethereum L2 **test network**
  (chain id 84532). It is not Base mainnet and not a production environment.
- All on-chain state is testnet state and may be reset, pruned or discarded.

## No real value

- **Test ETH** on Base Sepolia is a valueless testnet asset used only to pay gas.
- **MockUSDC / test USDC** used in the pilot is a test token with **no monetary
  value**. It is not USDC, not a stablecoin, and not redeemable for anything.
- Vault shares (`nbUSDC`) minted in the pilot are testnet artifacts with no value.

## Not a financial offering

- Nothing in this pilot is an offer, solicitation, recommendation or sale of any
  security, financial instrument, insurance product or investment.
- There is **no promise, projection or guarantee of any return, yield or profit**.
  Any numbers shown are for technical demonstration only.
- Participation does not create any client, advisory, insurance or fiduciary
  relationship.

## Purpose is technical validation

- The pilot exists to validate onboarding, KYB, role assignment, portfolio and
  claim flows, and institutional UX on testnet.
- Data you submit (e.g. KYB form fields) is processed for pilot operation and
  technical validation. Do not submit secrets, credentials or real sensitive
  personal data beyond what the pilot requires.

## Production requires separate review

- Any move toward Base mainnet or any production / real-funds use is **out of
  scope** for this pilot and requires a separate, independent legal, compliance
  and security review.
- Governance hardening (key separation / deployer hand-off, external audit) is
  not part of this pilot.

The pilot software is provided "as is", without warranties of any kind, to the
extent permitted by law. Participants use it at their own risk on testnet.
